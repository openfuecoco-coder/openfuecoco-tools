HTML Entity Encoder & Decoder

Convert special characters to HTML entities to prevent XSS or render literal characters in HTML.

Guide & Documentation

How to Use

1. Paste your text into the Input Text area.
2. Click Encode to convert characters like <, >, & into HTML entities (e.g., &lt;).
3. Click Decode to convert HTML entities back into their original characters.

Example Input

Input: <div>Hello & Welcome!</div> Output (Encoded): &lt;div&gt;Hello &amp; Welcome!&lt;/div&gt;

Common Mistakes

  • Double Encoding: Encoding an already encoded string will result in &amp;lt;. Always decode first if unsure.
  • Case Sensitivity: While most named entities are case-sensitive (e.g., &nbsp;), decimal and hex codes are usually handled by the browser regardless of case.
