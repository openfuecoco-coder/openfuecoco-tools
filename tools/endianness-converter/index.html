Endianness Converter

Enter a hexadecimal string to swap byte order.

How to use

Enter a hex string (even length), click the button, and it will swap pairs of bytes (e.g. 12 34 56 78 becomes 78 56 34 12).

Common Mistakes

  • Forgetting to include all bytes (must be even length).
  • Misinterpreting bitwise endianness vs byte-level swapping.
