ASCII Table Reference

DecHexCharName

How to use

This table provides the decimal, hexadecimal, and character mappings for standard ASCII (0-127). It is essential for troubleshooting character encoding and serial communication protocols.

Example

  • Dec: 65 → Hex: 0x41 → Char: 'A'

Common mistakes

Do not confuse non-printable control characters (0-31) with visible characters. They are represented as 'Control' in this table.
