Markdown 表格產生器

請在下方輸入您的資料，每行代表一行，以逗號、分號或Tab鍵分隔各欄位。

使用說明

1. 在上方輸入框貼上或輸入您的資料。
2. 每行代表表格的一列。
3. 欄位之間請使用逗號 (,)、分號 (;) 或 Tab 鍵分隔。
4. 點擊「生成 Markdown 表格」按鈕即可在下方輸出框看到結果。

範例輸入

Name,Age,City
Alice,30,New York
Bob,24,London

範例輸出

| Name | Age | City |
|------|-----|------|
| Alice | 30 | New York |
| Bob | 24 | London |

常見錯誤

1. 欄位分隔符不一致可能導致表格格式混亂。
2. 確保所有行都有相同數量的欄位，否則表格可能無法對齊。
