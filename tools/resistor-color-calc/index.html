How to use

  1. Select the color for each of the four bands on your resistor using the dropdowns.
  2. The calculated resistance value (in Ohms) and its tolerance percentage will be displayed automatically.

Examples

  • Input: Brown (Band 1), Black (Band 2), Red (Multiplier), Gold (Tolerance) → Output: 1kΩ ± 5%
  • Input: Red (Band 1), Violet (Band 2), Orange (Multiplier), Silver (Tolerance) → Output: 27kΩ ± 10%

Common mistakes

Ensure you read the color bands in the correct order. The first band is usually closest to an end, and the tolerance band is often slightly separated or thicker.
