Text Case Converter

Tool Body

How to Use

  1. Enter your text into the "Enter your text here..." box.
  2. Click on any of the conversion buttons (UPPERCASE, lowercase, Title Case, Sentence case, tOgGlE cAsE) to transform your text.
  3. The converted text will appear in the "Converted text will appear here..." box, ready for you to copy.
  4. Use the "Clear" button to reset both text areas.

Example Input

Input: Hello world, this is a test.

Output (UPPERCASE): HELLO WORLD, THIS IS A TEST.

Output (Title Case): Hello World, This Is A Test.

Common Mistakes

  • Forgetting to copy the output text before clearing the fields.
  • Expecting complex formatting (like bold, italics) to be preserved; this tool only handles character casing.
  • Not understanding "Title Case" vs. "Sentence Case": Title Case capitalizes the first letter of each major word, while Sentence Case only capitalizes the first letter of the sentence.
