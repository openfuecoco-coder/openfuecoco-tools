How to Use

1. Paste the full URL (including the ? and parameters) into the input field.

2. The tool will automatically parse the query string and display the keys and values in the table.

3. Parameters are automatically URL-decoded (e.g., %20 becomes a space).

Example Input

https://shop.example.com/search?category=electronics&sort=price_asc&q=smartphone&lang=en-US

Expected Result:

  • category: electronics
  • sort: price_asc
  • q: smartphone
  • lang: en-US

Common Mistakes

  • Omitting the ?: Ensure the URL contains a query string starting with a question mark.
  • Fragment Identifiers: Note that parameters after a # (fragment) are not considered part of the query string by standard URL parsing.
  • Encoding Issues: If you see % signs in the results, it means the input was double-encoded.
