Common Mistakes

  • Confusing Decoding with Verification: This tool only decodes the token. It does NOT verify the signature. A decoded token is not necessarily a valid or trusted token.
  • Wrong Format: Ensure you are pasting a valid JWT. A JWT should have three parts separated by dots (.).
  • Including "Bearer " prefix: If you copied the token from an Authorization header, remove the word Bearer before pasting.
