How to Use

        
Paste your JSON Web Token (JWT) into the text area above and click "Decode". The tool will split the token into its three components (Header, Payload, and Signature) and decode the Base64Url encoded JSON parts.

Common Mistakes

        
    
            
  • Trying to decode the signature: The signature is a cryptographic hash, not encoded JSON. It cannot be "decoded" to text.
    • 
            
  • Invalid Format: Ensure the token contains exactly two dots (three parts). If it's missing a part, it's not a valid JWT.
    • 
            
  • Wrong Base64 encoding: This tool uses Base64Url decoding. Standard Base64 decoders might fail if the token contains '-' or '_'.
