# CSV to JSON Converter

此工具用于将逗号分隔值 (CSV) 格式的数据转换为 JavaScript 对象表示法 (JSON) 格式。

## 工作流程

1.  用户提供 CSV 格式的输入数据。
2.  工具解析 CSV 数据，识别标题行（如果存在）和数据行。
3.  将解析后的数据结构化为 JSON 格式。
4.  返回 JSON 格式的输出。
