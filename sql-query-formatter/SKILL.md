# SQL Query Formatter

This tool helps to format SQL queries, making them more readable and maintainable.

## Workflow

1.  User provides an SQL query.
2.  The tool parses and formats the query according to common SQL formatting standards (e.g., indentation, capitalization).
3.  Returns the formatted SQL query.
